Lo primero es registrarse e ingresar con sus credenciales para comenzar a experimentar con tus propias credenciales.

Ahora debemos seleccionar en el buscador la opcion EC2.

Ahora debemos dar clic en Launch.

Paso 1 debemos dar clic en 64-bit.

Paso2 debemos dar clic en T2.

Paso 3 debemos cambiar los valores indicados en la imagen.

Paso 4 se nos asignan 8 gigas de espacio y damos clic en next.

Paso 5 a√±adimos la etiqueta Name y su respectivo valor.

Paso 6 debemos asignar un grupo de Seguridad.

Paso 7 revisamos y continuamos dando clic en Next.

Paso 7.1 seleccionamos proceder sin Llave.

Ahora debemos dar clic en el Enlace para continuar.

Esperamos a que el estado diga Running y damos a Conectar.

Instalamos Node con los siguientes pasos: sudo yum update.

Luego confirmamos

Instalamos el controlador de versiones NVM: curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | bash.

Instalar la ultima version de Node: . ~/.nvm/nvm.sh.

Instalar la ultima version de Node: nvm install node.

Comprobamos que este instalado correctamente: node -e "console.log('Running Node.js ' + process.version)".

Ahora instalaremos Git: sudo yum install git.

Clonaremos el codigo del proyecto Chat: git clone https://github.com/abkunal/Chat-App-using-Socket.io.git Y despues ingresaremos a la carpeta que contiene el proyecto con: cd Chat-App-using-Socket.io.

Se instalan las librerias Node: npm install

Y luego se instala las librerias PM2: npm install -g pm2

Ahora ejecutaremos el PM2: pm2 start app.js

y probaremos el PM2: curl 127.0.0.1:5000

Debemos obtener la IP publica desde nuestra Instancia: Dentro de Seguridad
